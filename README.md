# K2 DarkMode - A JavaScript Dark Mode Plugin

## Features
    * Easy & Lightweight (~6kb)
    * Time can be added for Automatic Applying Dark Mode
    * Very Easy to Customize as per your needs - Only CSS needs to be updated.
